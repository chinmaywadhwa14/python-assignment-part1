Student Grade Tracker – Python Assignment

Overview:
This is a Python assignment where I worked on basic concepts like loops, conditionals, lists, and string manipulation. The main aim was to apply these concepts in a practical way.

Tasks Completed:

Task 1: Data Parsing and Profile Cleaning  
In this task, I cleaned student data by formatting names properly, converting roll numbers into integers, and converting marks into a list. I also added validation to ensure names are correct.

Task 2: Marks Analysis  
Here, I assigned grades using conditions, calculated total and average marks, and identified the highest and lowest scoring subjects. I also included a feature to add new subjects with proper input validation.

Task 3: Class Performance Summary  
In this part, I calculated the average marks for each student and assigned pass or fail status. I also counted the number of students who passed and failed, and calculated the overall class average.

Task 4: String Manipulation  
This task involved working with text. I cleaned the given paragraph, converted it to title case, counted occurrences of "python", replaced it with "Python 3", and printed the sentences in a numbered format.

This assignment helped me understand how different Python concepts can be combined to solve practical problems.
